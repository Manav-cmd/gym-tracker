# 🏋️ Exercise Tracker (Java Web App)

A simple web-based **Exercise Tracking Application** built using **Java Web Technologies**.
This project demonstrates how a Java backend interacts with a database and a web interface to manage workout records.

🌐 **Live Demo:**
[https://exercise-tracker.up.railway.app/](https://exercise-tracker.up.railway.app/)

---

# 📌 Project Overview

The Exercise Tracker allows users to record and manage their workout activities.
It demonstrates the **end-to-end flow of a Java web application**:

Browser → Servlet → Business Logic → Database → Response

The application focuses on understanding how backend components interact with the presentation layer and the database.

---

# 🚀 Features

* Add new exercise records
* View stored workouts
* Update exercise information
* Delete workout entries
* Simple web interface for managing data
* Persistent storage using a relational database
